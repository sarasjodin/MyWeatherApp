My Bootstrap Weather App

JavaScript code

To make the app interactive, I implemented JavaScript functionality.

This allowed me to add dynamic features such as fetching real-time weather data from the OpenWeather API. With JavaScript, I could handle user input, perform conversions between Fahrenheit and Celsius, and update the user interface dynamically to display accurate and up-to-date weather information.

Bootstrap framework

Create an app with or without a framework? That's an interesting question!

I'm quite used to create websites without using any framework, therefore I wanted to dig into Bootstrap for this app. Bootstrap provided me with pre-built components and styles, making it easier to create responsive layouts, consistent buttons, forms, and navigation elements across different devices.

The search engine

The search engine is an essential JavaScript functionality that deserves a separate mention.

It empowers users to input a particular city or location and obtain the corresponding weather information. This feature must be programmed in a user-friendly manner to enhance the overall usability of the app.

Read More
Three themes

Lastly, I incorporated the concept of themes into the app: The classic options plus "a creative version".

Users have three themes to choose from, including the classic options of light and dark. The creative theme is designed to introduce an element of surprise and playfulness to the app. When users select the creative theme, a unique feature comes into play: each time they press the creative theme button, the app generates and displays a random color. This means that the color scheme of the app dynamically changes with each press, offering a fresh and unexpected visual experience. The intention behind the creative theme's random color functionality is to engage users, spark curiosity, and create a sense of exploration within the app. It offers a playful twist to the visual customization options, ensuring that each interaction with the creative theme feels unique and brings a fresh burst of color to the user's experience.

I hope this explanation helps you understand the work that goes into building a weather app from a developer's point of view.
